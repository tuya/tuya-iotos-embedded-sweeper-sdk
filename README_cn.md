# Tuya嵌入式扫地机SDKSDK

## 介绍

涂鸦扫地机 SDK 是指在涂鸦 IoTOS 系统中针对基于Linux系统开发的扫地机设备推出的软件开发包。提供标准 C 语言库文件，头文件以及对应的说明文档。
